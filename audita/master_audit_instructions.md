# Orouba Website Multi-Agent UX/UI Audit Orchestration

## Overview
This directory contains the necessary personas and report templates to conduct a comprehensive, multi-agent audit of the Orouba Food Industry website (https://orouba-five.vercel.app). 

By dividing the audit into 6 specialized personas, we ensure all aspects of the user experience, interface design, technical performance, accessibility, content, and conversion rate optimization are thoroughly evaluated.

## The Agents
1. **[Agent 1: Sarah (UX Lead)](./agent_1_ux_lead.md)** - Focuses on user journeys, B2B/B2C flows, and site architecture.
2. **[Agent 2: Alex (UI Designer)](./agent_2_ui_designer.md)** - Focuses on branding, typography, color, and RTL Arabic layouts.
3. **[Agent 3: Jordan (Accessibility)](./agent_3_accessibility.md)** - Focuses on WCAG compliance, screen readers, and keyboard navigation.
4. **[Agent 4: David (Performance & SEO)](./agent_4_performance.md)** - Focuses on Core Web Vitals, mobile responsiveness, and technical SEO.
5. **[Agent 5: Fatima (Content)](./agent_5_content.md)** - Focuses on Arabic copywriting, tone of voice, and localization.
6. **[Agent 6: Michael (CRO)](./agent_6_cro.md)** - Focuses on CTAs, lead generation friction, and trust signals.

## Execution Instructions
1. Assign each `.md` file to a corresponding AI agent (or team member).
2. Each agent must read their **Persona Profile**, visit the target URL, and execute the **Tasks & Checklist**.
3. Each agent must fill out the **Audit Report** section at the bottom of their respective file.
4. Once all 6 agents have completed their reports, **Agent 1 (Sarah - UX Lead)** should review all files and write a final **Executive Summary** synthesizing the most critical findings and prioritizing the recommendations.
